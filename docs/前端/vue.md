## 安装

:::tip
前提是安装好了 npm，cnpm 或 yarn
:::

- 全局安装 vue-cli 。

```shell
yarn global add vue-cli
```

## 创建一个基于 webpack 模板的新项目

```shell
vue init webpack my-project
```

可参考 runoob 教程：[https://www.runoob.com/vue2/vue-tutorial.html](https://www.runoob.com/vue2/vue-tutorial.html)
