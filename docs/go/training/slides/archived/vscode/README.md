### 常用快捷键

- 展示所有命令：Ctrl + Shift + P
- 快速打开文件：Ctrl + P
- 查找：Ctrl + F
- 全局查找：Ctrl + Shift + F
- 多光标
    - Ctrl + Alt + Up/Down
    - Alt + 鼠标点击
- 选中同一单词(从上往下，一次一个)：Ctrl + D
- 选中所有同一单词：Ctrl + Shift + L
- 上下移动行：Alt + Up/Down
- 行缩进：Ctrl + [ 、 Ctrl + ]
- [所有默认快捷键](https://code.visualstudio.com/docs/getstarted/keybindings#_default-keyboard-shortcuts)
- Mac 下所有 Ctrl 替换为 CMD 键
