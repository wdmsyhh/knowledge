# 公网IP

大陆内服务器无法访问，香港是可以的，说明此IP在跨境访问时受限了，建议您换IP

1.将公网IP转换为弹性公网Ip（不可逆）

参考文档：https://help.aliyun.com/document_detail/61290.html

2、解绑这个弹性公网IP，之后重新申请购买一个弹性公网Ip，并绑定到服务器上，达到更换IP的需求。原来解绑下来的旧弹性公网IP如果用不到请及时释放，否则会产生费用
