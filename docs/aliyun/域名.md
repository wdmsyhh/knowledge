# 域名

## 域名购买

- 进入[阿里云官网](https://www.aliyun.com/)。

- 点击产品，搜索域名，选择合适的域名进行购买。

    ![yuming-search](./images/yuming-search.png)

- 点击工作台搜索域名进入域名列表

    ![](./images/gongzuotai.png)


## 域名解析（域名绑定服务器 IP）

选择买境外服务器，不需要备案

- 点击解析

    ![](./images/yu_ming_lie_biao.png)

- 点击添加记录

    ![](./images/tian_jia_ji_lu.png)
